1)Ek naya folder banao (e.g., Q2_StudentSystem) aur VS Code me open karo.
2)Terminal 1: mkdir backend, cd backend, npm init -y
3)New Terminal (Terminal 2): npm create vite@latest frontend -- --template react
4)Backend directory me ek file banao: server.js
5)Terminal 1 (Backend): npm install express mongodb cors
6)Neeche diye gaye codes ko in files me update karo:
-backend/server.js
-frontend/src/App.css
-frontend/src/App.jsx
7)Terminal 1 (Backend): node server.js
8)Terminal 2 (Frontend): cd frontend, npm install, aur npm run dev
