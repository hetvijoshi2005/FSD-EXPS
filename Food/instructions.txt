1) Ek naya folder banao (e.g., Q1a_FoodOrder) aur VS Code me open karo.

2) Terminal 1: mkdir backend, cd backend, npm init -y

3) New Terminal (Terminal 2): npm create vite@latest frontend -- --template react

4) Backend directory me ek file banao: server.js

5) Terminal 1 (Backend): npm install express cors (Dhyan do: MongoDB install nahi kar rahe kyuki minimal backend chahiye)

6) Neeche diye gaye codes ko in files me paste karo:

7) backend/server.js

8) frontend/src/App.css

9) frontend/src/App.jsx

10) Terminal 1 (Backend): node server.js (Run karke chhod do)

11) Terminal 2 (Frontend): cd frontend, npm install, aur phir npm run dev
